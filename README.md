# ELONTRADEX BROKER

Welcome to ELONTRADEX BROKER, a cutting-edge cryptocurrency trading platform built with Next.js, Firebase, and Genkit AI.

## Description

ELONTRADEX BROKER is a comprehensive financial application providing a secure and intuitive interface for managing digital assets.

### Key Features:

*   **Real-time Market Data:** Live price tracking for major cryptocurrencies via CoinGecko API.
*   **Secure Authentication:** Powered by Firebase Authentication.
*   **Portfolio Management:** Track your balance and transaction history in real-time using Cloud Firestore.
*   **AI-Powered Support:** Integrated support chatbot ("Xavier") built with Genkit.
*   **Admin Panel:** Comprehensive administrative tools for user management, withdrawal processing, and site-wide settings.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens using Tailwind CSS and Shadcn UI.

## Tech Stack

*   **Frontend:** Next.js 14 (App Router), React, Tailwind CSS, Shadcn UI.
*   **Backend:** Firebase (Authentication, Firestore).
*   **AI:** Genkit (Google Gemini 2.0).
*   **Icons:** Lucide React.

## Deployment Information

*   **Live Branch:** `main`
*   **App Root Directory:** `/`

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Dante1245/elontradexbroker.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
